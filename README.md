# Цифровой двойник модульной орбитальной станции

Веб-прототип 4-го этапа на Flask + Three.js: интерактивная 3D-визуализация модульной орбитальной станции со стыковкой, орбитальным движением, межспутниковой связью и настройкой параметров в реальном времени.

## Запуск

```bash
python3 -m venv .venv
.venv/bin/pip install -r requirements.txt
.venv/bin/python app.py
```

Открыть в браузере: `http://127.0.0.1:7001`.

Если нужен другой порт, его можно указать без изменения кода:

```bash
PORT=7010 .venv/bin/python app.py
```

## Что есть в проекте

- Многостраничный Flask-сайт:
  - `/` - главная страница с описанием проекта;
  - `/simulator` - 3D-симулятор цифрового двойника;
  - `/metrics` - метрики ценности и сценарии проверки;
  - `/architecture` - архитектура решения и источники открытых данных.
- 3D-сцена с орбитой и выбором небесного тела: Земля, Луна, Марс, Венера, Юпитер, Сатурн.
- Анимация стыковки и расширения станции новыми модулями.
- Визуализация межспутниковых радиолиний и телеметрии.
- Управление параметрами орбиты, ориентации станции, солнечных панелей, состава модулей и связи.
- Переключение интерфейса RU / EN в симуляторе.
- Блок с открытыми источниками данных и примером TLE.
